TP React – Création d’un composant fonctionnel : Owl.js

### Objectif

Apprendre à créer, personnaliser et utiliser un composant React fonctionnel, en manipulant les props et le rendu conditionnel.

### Consignes

1. **Créer un composant `Owl`**
   * Le composant doit afficher une image de hibou (owl) et un texte descriptif.
   * L’image doit être passée en prop au composant.
2. **Utiliser le composant dans une application React**
   * Importez et affichez le composant `Owl` dans votre application principale.
   * Passez différentes images et descriptions pour tester la réutilisabilité du composant.
3. **Bonus**
   * Ajouter un effet au survol de l’image.
   * Affichez un message différent si aucune image n’est fournie.

### Contraintes techniques

* Utilisez la syntaxe JSX.
* Respectez la structure d’un composant fonctionnel.
* Commentez votre code pour expliquer vos choix.

### Rendu attendu

* Un fichier `Owl.js` contenant le composant.
* Un exemple d’utilisation dans votre application principale.
* Un court rapport (5-10 lignes) expliquant votre démarche.
